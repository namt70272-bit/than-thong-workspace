Reusable workflow and template examples.

| File | Purpose |
| --- | --- |

## Use this folder for

- small runnable or inspectable examples
- starter templates for recurring workflows
- patterns worth copying and adapting

## Do not use this folder for

- production configs
- generated reports
- large imported reference dumps

## Maintenance

- Keep examples minimal and readable.
- Prefer one example per clear use case.
- Add short descriptions when a filename alone is not enough.
