# memory/README.md

Daily memory log directory.

## Purpose

Use this folder for dated notes that capture what happened during work.

## Current files

- `2026-05-11.md`
- `2026-05-12-awesome-skills-intake.md`

## Naming guidance

Preferred pattern:
- `YYYY-MM-DD.md` for standard daily logs
- `YYYY-MM-DD-topic.md` for focused notes attached to a date

## What belongs here

- notable decisions
- progress summaries
- reminders worth keeping for later sessions
- intake notes from audits, imports, or research

## What does not belong here

- polished long-term memory (`MEMORY.md` at root, if/when used)
- generated reports (`reports/`)
- imported references (`references/`)

## Maintenance

- Keep entries chronological.
- Archive old material instead of deleting.
- If the folder grows large, add `INDEX.md` for navigation.
