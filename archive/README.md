## Rules

- Prefer moving here over deleting when a file may still be useful.
- Keep archived files clearly labeled and dated when possible.
- If archived material clusters around a theme, group it in a dedicated subfolder.

## Not for

- active reports (`reports/`)
- live scripts (`scripts/`)
- current references (`references/`)

Archive is for material that is no longer primary but still worth keeping around.
