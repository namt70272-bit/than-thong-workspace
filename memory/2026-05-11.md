### Hook fix đã apply:
**Root cause:** Plugin dùng `api.on("before_agent_reply", (event, ctx) => {...})` nhưng OpenClaw 2026.5.2 hook API chỉ truyền **1 param `event`**, không có `ctx`. Nên `ctx.userMessage` luôn undefined → hook return sớm, không fire gì cả.

**Fix:**
1. Thay `before_agent_reply` bằng `before_prompt_build` cho **auto-recall**:
   - Hook nhận `event.messages` (session messages) → extract last user message
   - Search Qdrant → return `{ prependSystemContext: "..." }` để inject context vào system prompt
2. Thêm `agent_end` cho **auto-capture**:
   - Hook observation, nhận `event.messages` (final messages)
   - Extract last user message → check pattern match → store vào Qdrant
3. Thêm `hooks.allowConversationAccess: true` trong staged config (cần thiết cho `agent_end`)

**File modified:** `openclaw-plugin-js/index.js`
