# DOMAIN DOSSIERS

Mỗi file trong thư mục này là 1 hồ sơ cho 1 mảng hệ thống.

## Mẫu nội dung bắt buộc
- Mục tiêu của mảng
- Tương đương hiện có trong hệ thống hiện tại
- Nguồn từ `G:\Ai`
- Phân loại A0 / A1 / B
- Phần thực sự nên lấy
- Phần không nên lấy
- Cách nhập tối thiểu vào `E:`
- Checklist trước khi nối thật

## Quy ước file
- `01-agent-management.md`
- `02-work-management.md`
- ...
- `16-infra-platform.md`
