# A0 — Tích hợp API

## Đã xác nhận an toàn tuyệt đối để lấy trước

### 1) API-Mới / API-Tuyển / BộAPIBackend
**Chỉ lấy:**
- route naming
- endpoint grouping
- config/schema docs

### 2) FastAPI-MẫuFullStack
**Chỉ lấy:**
- route structure
- folder layout
- auth flow mẫu

### 3) Chat2API / Frontend-SDK
**Chỉ lấy:**
- proxy pattern
- frontend/backend contract ideas

**Không lấy:**
- server runtime
- proxy chạy thật

## Kết luận
A0 mảng API là lấy schema và contract pattern, chưa đưa backend mới vào hệ.
